A simple passport photo generator.



## Installation

Use this libary [pillow](https://pillow.readthedocs.io/en/stable/) to install the PassbildMaschine.

```bash
pip install PIL
```
You`ll also need [tkinter](https://docs.python.org/3/library/tkinter.html).
```bash
pip install tkinter
```
## Usage

```python
python Main.py
```
